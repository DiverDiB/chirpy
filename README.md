# Chirpy
## Description
A lightning-fast social media backend built in Go. This project demonstrates core HTTP server concepts, routing, and database interactions.

## Installation
1. Clone the repository.

2. Ensure you have Go installed.

3. Run `go build` to compile the project.

## Usage
1. Configure your environment variables in a `.env` file.

2. Start the server using `./chirpy`.

3. The API will be available at `http://localhost:8080`.

## Features
• RESTful API endpoints for chirps and users.

• Secure authentication and password hashing.

• Efficient data storage and retrieval.
